# 变更日志

本文档记录项目的所有功能变更和设计说明。

## [功能1] 初始化 Vue3 + TypeScript + Pinia 项目

**日期**: 2025-04-01

**设计说明**:
- 使用 Vite 创建 Vue3 + TypeScript 项目模板
- 安装核心依赖: Pinia (状态管理)、Monaco Editor (代码编辑)、@vue/repl (组件渲染)
- 安装辅助依赖: Splitpanes (分栏布局)、Iconify (图标)、JSZip (项目导出)
- 项目结构采用模块化设计，组件、状态、类型分离

**实现要点**:
- Vite 提供快速的开发和构建体验
- Pinia 作为 Vue 官方推荐的状态管理方案，支持 TypeScript
- Monaco Editor 是 VS Code 同款编辑器，功能强大
- @vue/repl 是 Vue 官方提供的 REPL 渲染组件

---

## [功能2] 实现基础布局和主题切换

**日期**: 2025-04-01

**设计说明**:
- 采用左中右三栏弹性布局，使用 Splitpanes 实现可拖拽调整的分栏
- 左侧固定宽度显示文件树
- 中间和右侧可灵活调整比例
- 主题系统使用 CSS 变量实现，支持亮/暗两种主题
- 主题状态存储在 Pinia 中，全局可访问

**实现要点**:
- Splitpanes 提供流畅的分栏拖拽体验
- CSS 变量定义在 :root 和 [data-theme="dark"] 下
- 使用 localStorage 持久化用户主题偏好
- 主题切换按钮置于右上角

---

## [功能3] 实现文件管理组件

**日期**: 2025-04-01

**设计说明**:
- 文件树采用递归组件结构，支持无限层级嵌套
- 支持文件夹和文件的新增、删除、重命名
- 右键菜单提供操作入口
- 当前选中文件高亮显示
- 文件内容存储在 Pinia store 中

**实现要点**:
- 递归 FileTree 组件渲染文件夹和文件
- 右键菜单使用绝对定位，点击外部自动关闭
- 文件名编辑使用 input 内联编辑
- 删除操作需要二次确认

---

## [功能4] 实现编码区域

**日期**: 2025-04-01

**设计说明**:
- 集成 Monaco Editor 作为代码编辑器
- 支持 Vue 语法高亮和代码提示
- 实时保存文件内容到 store
- 支持多文件 tab 切换
- 未保存文件显示标记

**实现要点**:
- Monaco Editor 使用 vite-plugin-monaco-editor 集成
- 配置 Vue 语言支持和代码格式化
- 使用防抖处理实时保存，避免频繁更新
- Tab 栏显示文件名和未保存状态

## [功能5] 实现预览区域

**日期**: 2025-04-01

**设计说明**:
- 使用 @vue/repl 提供的 Sandbox 组件渲染 Vue 组件
- 实时编译和渲染编辑器中的代码
- 支持手动刷新预览
- 支持全屏查看预览
- 错误信息显示在预览区

**实现要点**:
- 从 store 获取当前文件内容传给编译器
- 使用 iframe 隔离预览环境
- 刷新功能通过重新挂载组件实现
- 全屏功能使用 Fullscreen API

## [功能6] 实现依赖库配置

## [功能7] 实现项目导出功能

**日期**: 2025-04-01

**设计说明**:
- 支持一键导出项目为 ZIP 文件
- 导出包含所有文件、文件夹结构、依赖配置
- 生成可运行的项目结构
- 提供下载链接

**实现要点**:
- 使用 JSZip 生成 ZIP 文件
- 生成标准的 Vite + Vue3 项目结构
- 包含 package.json、vite.config.ts 等配置文件
- 使用 FileSaver.js 或原生方法触发下载
